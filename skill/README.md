# Viewflow skill for AI coding assistants

`SKILL.md` teaches an AI coding assistant how to build with Viewflow. It loads
the Viewflow conventions and points to the full docs at
`https://docs.viewflow.io/llms-full.txt`.

## Install (Claude Code)

Copy the skill into your skills directory:

```bash
mkdir -p ~/.claude/skills/building-with-viewflow
cp SKILL.md ~/.claude/skills/building-with-viewflow/
```

## Other tools

For Cursor, Copilot, or similar, point the assistant at
`https://docs.viewflow.io/AGENTS.md`, or paste it into your project rules file.
